# lawagents

Free open source toolkit for small firms ($2M–$50M annual revenue).

## Documentation

- **[Law Firm Roles & Skills](docs/law-firm-roles-and-skills.md)** — Roles, responsibilities, and agent-ready tasks for modern law firm structure.
- **[Role Architecture](roles/README.md)** — One folder per role with detailed instructions, routines, escalation paths, and agent-ready tasks.
- **[Practice Areas](practice-areas/README.md)** — Transactional (real estate, trademark, corporate) and litigation (PI, criminal, civil). Operational guidance, tools, and pain points from Reddit/X research.
- **[Law Firm Tools & Technologies](docs/law-firm-tools-and-technologies.md)** — Clio integrations (API + MCP), approval workflows, and guardrails. Branching point for other tools.

# lawagents

Free open source toolkit for small firms ($2M–$50M annual revenue).

## Documentation

- **[Law Firm Roles & Skills](docs/law-firm-roles-and-skills.md)** — Roles, responsibilities, and agent-ready tasks for modern law firm structure. Designed to pair with tools/technologies (MCP, APIs) for autonomous agent workflows with check-ins and guardrails.
- **[Law Firm Tools & Technologies](docs/law-firm-tools-and-technologies.md)** — Clio integrations (API + MCP), approval workflows, and guardrails. Branching point for other tools.

# Practice Area Architecture

Folder structure for attorney practice types: **transactional** and **litigation**. Each practice area includes operational guidance, tools, and current-state research from Reddit, X, and industry sources.

## Structure

| Type | Practice Areas | Link |
|------|----------------|------|
| **Transactional** | Overview, Real Estate, Trademark/IP, Corporate | [transactional/](transactional/) |
| **Litigation** | Overview, Personal Injury, Criminal Defense, Civil Litigation | [litigation/](litigation/) |

## Key Differences

| Aspect | Transactional | Litigation |
|--------|---------------|------------|
| **Focus** | Creating documents, closing deals | Resolving disputes, court proceedings |
| **Approach** | Collaborative (parties work toward common goal) | Adversarial (opposing sides seek advantage) |
| **Pace** | Steadier, intensifies near closing | Unpredictable, court-set deadlines |
| **Court** | Rarely | Frequently |
| **Client relationship** | Long-term | Intense but often shorter-term |

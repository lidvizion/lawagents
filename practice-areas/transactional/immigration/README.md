# Immigration

**Focus:** Visas, citizenship, asylum, removal defense. Forms-heavy, government portals, client counseling.

---

## What Immigration Attorneys Do

- **Nonimmigrant visas:** H-1B, L-1, O-1, E-2, B-1/B-2, F-1, J-1, etc.
- **Immigrant visas:** Family-based, employment-based, diversity lottery
- **Citizenship/naturalization:** N-400, citizenship applications
- **Asylum/refugee:** Asylum applications, refugee processing
- **Removal defense:** Deportation defense, cancellation of removal
- **Compliance:** I-9, worksite enforcement, employer compliance
- **Consular processing:** Visa applications at consulates abroad

---

## How to Operate

### 1. Forms and Government Portals

- **USCIS:** Online filing for many forms; case status tracking
- **Forms:** I-129, I-130, I-140, I-485, N-400, etc.—version control critical
- **Consular appointments:** NIV/IV scheduling; country-specific
- **DHS/DOJ portals:** Check case status; respond to RFEs

### 2. Client Counseling

- **Language:** Translation apps when needed; qualified interpreters for formal proceedings
- **Document collection:** Passports, birth certificates, marriage certs, employment letters
- **Timeline management:** Processing times vary; set expectations
- **Status updates:** USCIS processing times; client communication

### 3. Deadline and Compliance Tracking

- **Filing deadlines:** RFE response deadlines; court dates
- **Status expiration:** Visa expiration; extension filing
- **Compliance:** I-9 reverification; H-1B cap; PERM timelines

### 4. Document Management

- **Secure storage:** Client documents often include PII, sensitive data
- **Translation:** Certified translations when required
- **Evidence organization:** Application packages; exhibits

## Things to Keep in Mind

| Item | Why It Matters |
|------|----------------|
| **Processing times** | USCIS varies by form, office; manage expectations |
| **RFE deadlines** | Strict; often 30–90 days |
| **Unauthorized practice** | Immigration is federal; state bar + federal recognition |
| **Client vulnerability** | Many clients in precarious status; sensitivity |
| **Policy changes** | Administration changes affect rules; stay current |

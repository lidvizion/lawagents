# Employment / Labor (Transactional & Advisory)

**Focus:** Workplace law—hiring, firing, compliance, employment contracts, regulatory compliance (FLSA, anti-discrimination, etc.).

---

## What Employment/Labor Attorneys Do

- **Employment contracts:** Draft and negotiate employment agreements, non-competes, severance
- **Regulatory compliance:** FLSA, ADA, Title VII, FMLA, state wage/hour, anti-discrimination
- **Workplace policies:** Handbooks, PTO, remote work, harassment prevention
- **Investigations:** Internal investigations, EEOC/charges, DOL audits
- **Union/labor:** Collective bargaining, NLRB, union avoidance (management-side)
- **Advisory:** Day-to-day counseling on hiring, firing, discipline, accommodations

---

## How to Operate

### 1. Compliance Calendar

- **Deadline tracking:** EEOC response deadlines, DOL audits, state filing deadlines
- **Policy updates:** Annual handbook review; track regulatory changes
- **Training:** Harassment prevention, compliance training schedules

### 2. Document Workflow

- **Templates:** Employment agreements, separation agreements, policies
- **Version control:** Policy versions; track changes for audit
- **Client review:** HR/management review before implementation

### 3. Integration with Client Systems

- **HRIS data:** Workday, BambooHR, ADP—understand payroll/HR data for audits
- **Payroll:** Wage/hour compliance; overtime calculations; classification
- **Confidentiality:** Client HR data is sensitive; secure handling

---

## Things to Keep in Mind

| Item | Why It Matters |
|------|----------------|
| **FLSA** | Wage/hour, overtime, classification (exempt/non-exempt) |
| **EEOC charges** | Strict response deadlines; document everything |
| **State variations** | Wage, leave, non-compete laws vary by state |
| **At-will** | State-specific; exceptions (implied contract, public policy) |
| **Remote work** | Multi-state compliance; tax, employment law |

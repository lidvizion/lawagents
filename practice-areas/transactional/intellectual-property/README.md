## What IP Attorneys Do

- **Trademarks:** Applications, office actions, oppositions, renewals, licensing
- **Patents:** Applications, prosecution, prior art, licensing, opinions
- **Copyright:** Registration, licensing, DMCA, infringement
- **IP licensing:** Draft and negotiate licenses across all IP types
- **Infringement analysis:** Freedom to operate, validity, enforcement
- **Portfolio management:** Track filings, deadlines, renewals across portfolio

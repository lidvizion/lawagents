# Corporate (Transactional)

**Focus:** M&A, entity formation, contracts, commercial agreements, and corporate governance.

---

## What Corporate Attorneys Do

- **M&A:** Due diligence, deal structure, purchase agreements, closing
- **Entity formation:** LLCs, corporations, partnerships; operating agreements; bylaws
- **Contracts:** Commercial agreements, NDAs, vendor contracts, employment agreements
- **Financing:** Loan agreements, security documents, equity financing
- **Governance:** Board matters, shareholder agreements, compliance

---

## How to Operate

### 1. Deal Management

- **Deal stages:** LOI/term sheet → Due diligence → Definitive docs → Closing
- **Checklists:** Use transaction checklists for every deal type (acquisition, financing, formation)
- **Document control:** Central repository; version control; clear naming
- **Party coordination:** Buyer, seller, counsel, accountants, lenders

### 2. Due Diligence

- **Request list:** Standardized DD request list by deal type
- **Data room:** Organize documents; track received vs. pending
- **DD report:** Summarize findings; flag issues for client
- **Materiality:** Focus on material issues; avoid over-documenting minor items

### 3. Document Drafting and Negotiation

- **Templates:** Use firm-approved templates; customize for deal
- **Redlines:** Track all changes; document rationale for material changes
- **Comment resolution:** Log open points; resolve before closing
- **Ancillary documents:** Don't forget exhibits, schedules, certificates

### 4. Closing Mechanics

- **Closing checklist:** All conditions, documents, certificates, funds
- **Signature pages:** Coordinate execution; track who has signed
- **Closing deliverables:** Wire instructions, certificates, resolutions
- **Post-closing:** Filings, notifications, document retention

## Things to Keep in Mind

| Item | Why It Matters |
|------|----------------|
| **Material adverse change (MAC)** | Key reps and conditions; negotiate carveouts |
| **Indemnification** | Caps, baskets, survival; often heavily negotiated |
| **Conditions precedent** | Must be satisfied or waived before closing |
| **Regulatory approvals** | HSR, CFIUS, industry-specific |
| **Financing contingency** | Buyer's obligation to close may depend on funding |
| **Earnouts and escrows** | Post-closing mechanics; document clearly |

## Pain Points (Reddit, Industry)

- **Tool fragmentation:** Clio + Dropbox + DocuSign + Excel for deal tracking
- **Version control:** Multiple drafts across parties; "final_v7_FINAL" syndrome
- **Checklist adoption:** Deals close without formal checklists; items fall through cracks
- **Due diligence overload:** Volume of documents; need structure and prioritization
- **Scaling:** Midsize firms outgrow Tabs, Practice Master; Aderant/Elite expensive
- **Billing:** Corporate work often project/fixed fee; time tracking still needed for profitability

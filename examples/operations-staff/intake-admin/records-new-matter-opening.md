# Records Clerk / Case Opener: New Matter Opening

**Role:** [Records Clerk / Case Opener](../../../roles/records-clerk-case-opener/README.md)  
**Frequency:** As needed (when matter accepted)  
**Approval:** Attorney confirms matter details; Billing sets rates

---

## Scenario

Attorney has accepted a new matter after consultation. Records Clerk opens the matter in Clio, creates file structure, and coordinates handoff to Billing and Legal Assistant.

---

## Prerequisites

- Matter accepted by attorney
- Engagement letter signed (or in progress)
- Client and matter details confirmed
- Conflict check complete

---

## Step-by-Step

### 1. Matter Creation in Clio

| Step | Action | Tool |
|------|--------|------|
| 1.1 | Create new matter in Clio | Clio |
| 1.2 | Enter matter name (client + matter type, e.g., "Smith - PI MVA") | Clio |
| 1.3 | Add client contact(s) | Clio |
| 1.4 | Set practice area | Clio |
| 1.5 | Assign responsible attorney | Clio |
| 1.6 | Add matter description | Clio |

### 2. Billing Setup

| Step | Action |
|------|--------|
| 2.1 | Set billing arrangement (hourly, contingent, flat, hybrid) |
| 2.2 | Add matter billing codes if applicable |
| 2.3 | Coordinate with Billing Manager for rate setup |
| 2.4 | Add trust account if retainer received |

### 3. Document Setup

| Step | Action |
|------|--------|
| 3.1 | Create matter folder in document system |
| 3.2 | Upload engagement letter, retainer agreement |
| 3.3 | Create subfolder structure per firm standard |
| 3.4 | Add any intake documents from Legal Intake |

### 4. Task and Calendar

| Step | Action |
|------|--------|
| 4.1 | Add initial tasks (e.g., "Client intake call," "Conflict check") |
| 4.2 | Add key dates (SOL, contract deadline) if known |
| 4.3 | Assign tasks to Legal Assistant or attorney |

### 5. Handoff

| Step | Action |
|------|--------|
| 5.1 | Notify Legal Assistant that matter is ready for full setup |
| 5.2 | Notify Billing Manager for rate confirmation |
| 5.3 | Notify Legal Intake to update CRM (Accepted) |
| 5.4 | Close "Matter opening" task |

---

## Matter Opening Checklist (Printable)

- [ ] Matter created in Clio
- [ ] Client contacts added
- [ ] Billing arrangement set
- [ ] Engagement letter uploaded
- [ ] Folder structure created
- [ ] Initial tasks added
- [ ] Legal Assistant notified
- [ ] Billing Manager notified
- [ ] Legal Intake notified (CRM update)

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Matter creation | Attorney confirms matter details |
| Folder creation | Records Clerk or Legal Assistant |
| Billing setup | Billing Manager confirms rates |

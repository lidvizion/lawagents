# Legal Intake Specialist: Lead to Consultation

**Role:** [Legal Intake Specialist](../../../roles/legal-intake-specialist/README.md)  
**Frequency:** Per lead (typically within 24 hours of first contact)  
**Approval:** Attorney approves case acceptance; Intake schedules consultation

---

## Scenario

New lead comes in via phone, web form, or referral. Legal Intake Specialist qualifies the lead, gathers information, runs conflict check, and schedules initial consultation with an attorney.

---

## Step-by-Step

### 1. First Contact (Within 24 Hours)

| Step | Action |
|------|--------|
| 1.1 | Acknowledge lead (call back, email reply, or auto-respond) |
| 1.2 | Gather basics: name, contact, matter type, urgency |
| 1.3 | Determine if matter fits firm practice areas |
| 1.4 | If no fit: politely decline; provide referrals if appropriate |
| 1.5 | If fit: proceed to full intake |

### 2. Intake Information Gathering

| Step | Action | Tool |
|------|--------|------|
| 2.1 | Complete intake form with case details | Clio Grow, intake form |
| 2.2 | Collect party names (opposing party, related entities) | For conflict check |
| 2.3 | Gather key facts and potential value/urgency | Intake form |
| 2.4 | Request relevant documents if available | Email, portal |

### 3. Conflict Check

| Step | Action |
|------|--------|
| 3.1 | Run conflict check with party names |
| 3.2 | If potential conflict: escalate to attorney immediately |
| 3.3 | If clear: proceed to scheduling |

### 4. Case Criteria Evaluation

| Step | Action |
|------|--------|
| 4.1 | Evaluate against firm criteria (practice fit, profitability, capacity) |
| 4.2 | Apply lead scoring if firm uses |
| 4.3 | Document evaluation in CRM |
| 4.4 | If decline: send rejection letter (attorney approves); provide referrals |

### 5. Consultation Scheduling

| Step | Action |
|------|--------|
| 5.1 | Identify appropriate attorney for consultation |
| 5.2 | Check attorney availability |
| 5.3 | Offer 2–3 time slots to lead |
| 5.4 | Confirm appointment; send confirmation and reminder |
| 5.5 | Add to CRM; update pipeline status to "Consultation Scheduled" |

### 6. Pre-Consultation Prep

| Step | Action |
|------|--------|
| 6.1 | Prepare intake summary for attorney |
| 6.2 | Attach any collected documents |
| 6.3 | Send prep packet to attorney 24 hours before |
| 6.4 | Remind lead 24 hours before (email or text) |

---

## CRM Pipeline Statuses

| Status | Meaning |
|--------|---------|
| New Lead | First contact; not yet qualified |
| Qualified | Fits criteria; conflict clear |
| Consultation Scheduled | Appointment set |
| Consulted | Meeting held; awaiting decision |
| Accepted | Matter opened |
| Declined | Not taking; documented reason |

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Intake form completion | Attorney reviews before matter opening |
| Conflict pre-screening | Attorney confirms before engagement |
| CRM data entry | Follow firm data standards |
| Consultation scheduling | Confirm attorney availability |

---

## Tools

- **Clio Grow** — CRM, intake forms, pipeline
- **Conflict check** — Firm system or Clio
- **Calendar** — Attorney availability, scheduling

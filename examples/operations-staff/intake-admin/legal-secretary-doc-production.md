# Legal Secretary: Document Production Package

**Role:** [Legal Secretary](../../../roles/legal-secretary/README.md)  
**Frequency:** As needed (when producing documents to opposing counsel)  
**Approval:** Attorney approves before service

---

## Scenario

Paralegal or attorney has identified documents for production. Legal Secretary prepares the production package: Bates numbering, cover letter, and service.

---

## Prerequisites

- Documents identified for production
- Bates range assigned (or Legal Secretary assigns)
- Attorney has approved production

---

## Step-by-Step

### 1. Document Organization

| Step | Action |
|------|--------|
| 1.1 | Receive document set from paralegal or attorney |
| 1.2 | Verify document order (chronological or by category per instructions) |
| 1.3 | Ensure no privileged documents included |
| 1.4 | Confirm page count |

### 2. Bates Numbering

| Step | Action |
|------|--------|
| 2.1 | Apply Bates numbers (e.g., SMITH-000001 through SMITH-000150) |
| 2.2 | Use firm convention (matter prefix, sequential) |
| 2.3 | Ensure each page has Bates stamp |
| 2.4 | Create production log (Bates range, document description) |

### 2b. If Using E-Discovery Tool

| Step | Action |
|------|--------|
| 2b.1 | Coordinate with Document & Litigation Support |
| 2b.2 | Export production set with Bates from tool |
| 2b.3 | Verify output |

### 3. Cover Letter

| Step | Action |
|------|--------|
| 3.1 | Draft cover letter per firm template |
| 3.2 | Include: matter, production description, Bates range, date |
| 3.3 | Attorney reviews and signs |
| 3.4 | Attach production log |

### 4. Final Assembly

| Step | Action |
|------|--------|
| 4.1 | Combine cover letter, production log, and documents |
| 4.2 | Convert to PDF if electronic production |
| 4.3 | Burn to disc or prepare electronic delivery per agreement |
| 4.4 | Label media (matter name, date, Bates range) |

### 5. Service

| Step | Action |
|------|--------|
| 5.1 | Serve per discovery agreement (email, mail, portal) |
| 5.2 | Obtain proof of delivery |
| 5.3 | File proof of service if required |
| 5.4 | Save copy to matter file; update Clio |

---

## Production Log Format (Example)

| Bates Range | Description |
|-------------|-------------|
| SMITH-000001–000025 | Email chain, Smith to Jones, 1/15/2024 |
| SMITH-000026–000030 | Invoice, Acme Corp, 2/1/2024 |
| ... | ... |

---

## Tools

- **Adobe Acrobat / PDF tool** — Bates stamping
- **Word** — Cover letter
- **Clio** — Matter documents, tasks
- **E-discovery** — If production from review platform

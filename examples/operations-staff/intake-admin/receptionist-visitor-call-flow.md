# Receptionist: Visitor and Call Flow

**Role:** [Receptionist / Front Desk](../../../roles/receptionist-front-desk/README.md)  
**Frequency:** Continuous (every visitor, every call)  
**Approval:** Low—routine front-desk operations

---

## Scenario

Visitor arrives or call comes in. Receptionist greets, determines purpose, and routes appropriately while maintaining a professional first impression.

---

## Visitor Flow

### 1. Greeting

| Step | Action |
|------|--------|
| 1.1 | Greet visitor warmly and professionally |
| 1.2 | Ask: "How may I help you today?" |
| 1.3 | Determine purpose: appointment, delivery, walk-in, other |

### 2. Appointment Arrival

| Step | Action |
|------|--------|
| 2.1 | Confirm name and appointment time |
| 2.2 | Ask visitor to sign in (visitor log) |
| 2.3 | Issue visitor badge if required |
| 2.4 | Notify attorney or staff: "Your 10:00 appointment has arrived" |
| 2.5 | Offer seat, water; ensure timely pickup |

### 3. Walk-In (No Appointment)

| Step | Action |
|------|--------|
| 3.1 | Ask purpose of visit |
| 3.2 | If potential client: gather name, contact, brief matter type |
| 3.3 | Route to Legal Intake Specialist (or take message for callback) |
| 3.4 | If other (delivery, vendor): handle per firm protocol |
| 3.5 | Do not provide legal advice |

### 4. Delivery

| Step | Action |
|------|--------|
| 4.1 | Accept package; sign if required |
| 4.2 | Log delivery (date, sender, recipient) |
| 4.3 | Notify recipient; place in mail area or deliver to desk |

---

## Call Flow

### 1. Answer

| Step | Action |
|------|--------|
| 1.1 | Answer within 3 rings (or firm standard) |
| 1.2 | Use firm greeting: "Good morning, [Firm Name], how may I help you?" |
| 1.3 | Listen for purpose of call |

### 2. Routing

| Step | Action |
|------|--------|
| 2.1 | If caller asks for specific person: transfer to extension |
| 2.2 | If voicemail: take message; include name, number, brief message, urgency |
| 2.3 | If potential client: route to Legal Intake or take intake info for callback |
| 2.4 | If existing client: verify matter; route to responsible attorney or assistant |
| 2.5 | If vendor/other: route per firm protocol |

### 3. Message Taking

| Step | Action |
|------|--------|
| 3.1 | Write down: caller name, phone, date/time, message |
| 3.2 | Note urgency if indicated |
| 3.3 | Deliver message promptly (email, Slack, or written note) |
| 3.4 | Do not provide legal advice or case information |

### 4. When Unable to Help

| Step | Action |
|------|--------|
| 4.1 | Offer to take message |
| 4.2 | Offer callback time if appropriate |
| 4.3 | Never say "attorney is busy" — use "in a meeting" or "unavailable" |
| 4.4 | Maintain professional, courteous tone |

---

## Do Not

- Provide legal advice
- Discuss case details in common areas
- Leave visitors unattended for extended periods
- Give out attorney cell numbers without permission
- Promise callbacks without checking availability

---

## Escalation

| Situation | Action |
|-----------|--------|
| Angry or upset caller | Stay calm; offer to have supervisor or attorney callback |
| Emergency (e.g., client in custody) | Notify attorney immediately |
| Security concern | Notify Office Manager or COO |
| Media inquiry | Route to Managing Partner or designated contact |

---

## Tools

- **Phone system** — Transfer, voicemail
- **Visitor log** — Paper or digital
- **Calendar** — Check appointment
- **CRM / Clio** — If looking up client for routing

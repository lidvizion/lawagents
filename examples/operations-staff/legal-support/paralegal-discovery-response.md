# Paralegal: Discovery Response Workflow

**Role:** [Paralegal](../../../roles/paralegal/README.md)  
**Frequency:** As needed (when discovery received)  
**Approval:** Attorney reviews all substantive work before service

---

## Scenario

Opposing counsel serves Requests for Production (RFPs). Paralegal drafts responses, objections, and privilege log; attorney reviews and approves before service.

---

## Prerequisites

- Discovery received and logged
- Attorney assigns paralegal to draft response
- Document collection and review underway (if needed)

---

## Step-by-Step

### 1. Intake and Calendar

| Step | Action |
|------|--------|
| 1.1 | Log discovery in matter file; note due date |
| 1.2 | Add response deadline to calendar (typically 30 days; check rules) |
| 1.3 | Confirm court rules and local rules for format, timing |
| 1.4 | Create task in Clio: "Draft RFP responses" |

### 2. Document Collection and Review

| Step | Action |
|------|--------|
| 2.1 | Identify custodians and sources per request |
| 2.2 | Collect responsive documents (client, firm files) |
| 2.3 | Review for responsiveness, privilege, relevance |
| 2.4 | Flag privileged documents for privilege log |
| 2.5 | Organize responsive documents for production (Bates numbering) |

### 3. Draft Responses and Objections

| Step | Action |
|------|--------|
| 3.1 | Draft response to each request (admit, deny, object, produce) |
| 3.2 | Draft objections where appropriate (overbroad, privilege, etc.) |
| 3.3 | Use firm templates and prior responses for consistency |
| 3.4 | Prepare privilege log for withheld documents |
| 3.5 | Prepare production log (Bates range, description) |

### 4. Attorney Review

| Step | Action |
|------|--------|
| 4.1 | Submit draft to supervising attorney |
| 4.2 | Incorporate attorney revisions |
| 4.3 | Obtain final approval before service |

### 5. Production and Service

| Step | Action |
|------|--------|
| 5.1 | Coordinate with Document & Litigation Support for Bates numbering and format |
| 5.2 | Prepare cover letter; attach response, privilege log, production |
| 5.3 | Serve per court rules (e-file, mail, etc.) |
| 5.4 | File proof of service if required |
| 5.5 | Document in matter file; close task |

---

## Checklist (Attorney Approval Required)

- [ ] Responses drafted
- [ ] Objections drafted
- [ ] Privilege log prepared
- [ ] Production log prepared
- [ ] Attorney review complete
- [ ] Served and filed

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Discovery response drafts | Attorney reviews before service |
| Privilege log | Attorney confirms privilege designations |
| Production organization | Attorney approves before production |

---

## Tools

- **Clio** — Tasks, deadlines, matter documents
- **E-discovery** (Everlaw, etc.) — Document review, tagging, production
- **Word** — Response formatting, privilege log
- **E-filing** — Service and filing

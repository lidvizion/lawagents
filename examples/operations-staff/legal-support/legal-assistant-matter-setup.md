# Legal Assistant: Matter Setup and Calendar

**Role:** [Legal Assistant](../../../roles/legal-assistant/README.md)  
**Frequency:** As needed (new matter opened)  
**Approval:** Low—routine; attorney confirms matter details

---

## Scenario

New matter has been approved. Legal Assistant sets up matter folder, calendar, and task templates in Clio so the team can begin work.

---

## Prerequisites

- Matter approved by attorney/partner
- Records Clerk has opened matter in Clio (or Legal Assistant does it)
- Matter number and name confirmed

---

## Step-by-Step

### 1. Matter Configuration in Clio

| Step | Action | Tool |
|------|--------|------|
| 1.1 | Confirm matter exists; add matter details (description, practice area) | Clio |
| 1.2 | Add matter participants (attorneys, paralegal, client contacts) | Clio |
| 1.3 | Set billing arrangement (hourly, contingent, flat) | Clio |
| 1.4 | Apply task template if firm uses (e.g., PI template, real estate template) | Clio |

### 2. Folder Structure

| Step | Action |
|------|--------|
| 2.1 | Create matter folder in document system (Clio, SharePoint, etc.) |
| 2.2 | Create subfolders per firm standard (e.g., Pleadings, Discovery, Correspondence, Client Docs) |
| 2.3 | Add any initial documents (engagement letter, retainer) |

### 3. Calendar and Deadlines

| Step | Action |
|------|--------|
| 3.1 | Add key dates from engagement (e.g., SOL, contract deadline) |
| 3.2 | Add court deadlines if known |
| 3.3 | Sync matter calendar with attorney calendars |
| 3.4 | Set reminder for first substantive deadline |

### 4. Task Setup

| Step | Action |
|------|--------|
| 4.1 | Create initial tasks from template (e.g., "Conflict check," "Client intake call") |
| 4.2 | Assign tasks to appropriate team members |
| 4.3 | Set due dates where applicable |

### 5. Client Communication Setup

| Step | Action |
|------|--------|
| 5.1 | Add client to matter; verify contact info |
| 5.2 | Set up client portal access if firm uses |
| 5.3 | Prepare welcome letter or first-contact template for attorney |

### 6. Handoff to Team

| Step | Action |
|------|--------|
| 6.1 | Notify supervising attorney that matter is ready |
| 6.2 | Notify paralegal if assigned |
| 6.3 | Add matter to team meeting or status report |

---

## Matter Setup Checklist (Printable)

- [ ] Matter configured in Clio
- [ ] Participants added
- [ ] Billing arrangement set
- [ ] Folder structure created
- [ ] Key deadlines on calendar
- [ ] Initial tasks created and assigned
- [ ] Client contact added
- [ ] Team notified

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Matter folder creation | Attorney confirms matter details |
| Task template application | Legal Assistant or attorney assigns |
| Calendar entries | Attorney confirms substantive deadlines |

---

## Tools

- **Clio** — Matters, tasks, calendar, documents
- **Google Drive / SharePoint** — Document folders (if separate)

# Document & Litigation Support: E-Filing Preparation

**Role:** [Document & Litigation Support Specialist](../../../roles/document-litigation-support-specialist/README.md)  
**Frequency:** As needed (when filing with court)  
**Approval:** Attorney confirms before filing

---

## Scenario

Attorney has approved a pleading or motion for filing. Document & Litigation Support Specialist prepares the document per court rules and submits via e-filing system.

---

## Prerequisites

- Document finalized and approved by attorney
- Court and case number identified
- E-filing system access (e.g., PACER, state e-filing portal)

---

## Step-by-Step

### 1. Court Rules Check

| Step | Action |
|------|--------|
| 1.1 | Confirm court (federal, state, local) and division |
| 1.2 | Review court rules for formatting (margins, font, line spacing) |
| 1.3 | Review rules for document type (motion, opposition, reply) |
| 1.4 | Check page limits, exhibit requirements |
| 1.5 | Verify filing deadline and time zone |

### 2. Document Formatting

| Step | Action |
|------|--------|
| 2.1 | Apply court-required formatting (e.g., 1" margins, 12pt font) |
| 2.2 | Add caption, case number, document title |
| 2.3 | Number pages; add line numbers if required |
| 2.4 | Prepare exhibits with proper labels and tabs |
| 2.5 | Convert to PDF; ensure searchable and compliant |

### 3. E-Filing System Prep

| Step | Action |
|------|--------|
| 3.1 | Log into e-filing system |
| 3.2 | Select case; verify party information |
| 3.3 | Select document type (e.g., Motion, Memorandum) |
| 3.4 | Upload PDF(s) |
| 3.5 | Enter required metadata (title, description) |
| 3.6 | Add service contacts (opposing counsel) |

### 4. Attorney Final Review

| Step | Action |
|------|--------|
| 4.1 | Send draft filing package to attorney for final review |
| 4.2 | Incorporate any last-minute changes |
| 4.3 | Obtain attorney approval to file |

### 5. Filing and Service

| Step | Action |
|------|--------|
| 5.1 | Submit via e-filing system |
| 5.2 | Confirm filing accepted; obtain confirmation number |
| 5.3 | Download filed-stamped copy |
| 5.4 | Serve on opposing counsel per rules (often automatic via e-filing) |
| 5.5 | File proof of service if required |
| 5.6 | Save filed copy to matter file; update Clio |

### 6. Post-Filing

| Step | Action |
|------|--------|
| 6.1 | Add hearing date to calendar if set |
| 6.2 | Notify attorney of confirmation |
| 6.3 | Document in matter file |

---

## Court Rules Checklist (Common)

- [ ] Margins (often 1")
- [ ] Font (often 12pt Times or similar)
- [ ] Line spacing (often double)
- [ ] Page limits
- [ ] Caption format
- [ ] Exhibit format
- [ ] Signature block
- [ ] Certificate of service

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Filing checklist | Attorney confirms before filing |
| Format verification | Doc Support owns; attorney approves content |

## Tools

- **E-filing** — PACER/CM/ECF (federal), state portals (e.g., Tyler, InfoTrack)

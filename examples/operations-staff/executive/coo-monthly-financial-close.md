# COO: Monthly Financial Close

**Role:** [COO / Firm Administrator](../../../roles/coo-firm-administrator/README.md)  
**Frequency:** Monthly (typically 1st–5th business day)  
**Approval:** Managing Partner reviews before partnership presentation

---

## Scenario

End-of-month close. Produce financial reports, reconcile trust, and present to partnership.

---

## Prerequisites

- Billing Manager has completed invoice run and AR is current
- Finance/Accounting Manager has closed general ledger
- All time entries for prior month are submitted and approved

---

## Step-by-Step

### 1. Pre-Close Checklist (Day 1)

| Step | Action | Tool / Owner |
|------|--------|--------------|
| 1.1 | Confirm billing close complete | Billing Manager |
| 1.2 | Confirm GL close complete | Finance/Accounting Manager |
| 1.3 | Reconcile trust accounts | Billing or Finance |
| 1.4 | Verify payroll posted | Finance/Accounting |
| 1.5 | Collect department variance notes | COO |

### 2. Financial Report Generation (Day 2)

| Step | Action | Output |
|------|--------|--------|
| 2.1 | Run P&L for prior month | Monthly P&L |
| 2.2 | Run balance sheet snapshot | Balance sheet |
| 2.3 | Run cash flow summary | Cash flow |
| 2.4 | Run AR aging (if not in billing report) | Aging report |
| 2.5 | Run matter profitability summary | Realization, leverage |
| 2.6 | Run utilization by attorney | Utilization report |

### 3. Variance Analysis (Day 2–3)

| Step | Action |
|------|--------|
| 3.1 | Compare actual to budget by category |
| 3.2 | Note significant variances (>10% or threshold) |
| 3.3 | Document explanations (seasonal, one-time, trend) |
| 3.4 | Flag items for partnership discussion |

### 4. Partnership Presentation Prep (Day 3–4)

| Step | Action |
|------|--------|
| 4.1 | Draft presentation or memo (P&L, cash, AR, KPIs) |
| 4.2 | Add variance commentary |
| 4.3 | Include action items (collections, write-offs, budget) |
| 4.4 | Send draft to Managing Partner for review |

### 5. Managing Partner Review (Day 4)

| Step | Action |
|------|--------|
| 5.1 | Managing Partner reviews draft |
| 5.2 | Incorporate feedback |
| 5.3 | Finalize presentation |

### 6. Partnership Meeting (Day 5 or scheduled)

| Step | Action |
|------|--------|
| 6.1 | Present financial summary |
| 6.2 | Discuss variances and action items |
| 6.3 | Document decisions and follow-ups |
| 6.4 | Distribute final report to partners |

---

## Handoffs

| From | To | Item |
|------|-----|------|
| Billing Manager | COO | Invoice run complete, AR summary |
| Finance/Accounting | COO | GL close, payroll confirmation |
| COO | Managing Partner | Draft presentation |
| COO | Partnership | Final report, presentation |

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Financial report generation | Managing Partner reviews before partnership |
| Variance analysis | COO owns; escalate anomalies to Managing Partner |
| Presentation draft | Managing Partner approves before distribution |

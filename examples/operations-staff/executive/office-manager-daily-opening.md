# Office Manager: Daily Opening Checklist

**Role:** [Office Manager](../../../roles/office-manager/README.md)  
**Frequency:** Daily (before 9:00 AM or firm opening)  
**Approval:** Low—routine operations

---

## Scenario

Ensure the office is ready for the day: coverage, supplies, facilities, and staff coordination.

---

## Step-by-Step

### 1. Coverage Check (15 min before open)

| Step | Action |
|------|--------|
| 1.1 | Review who is in/out (PTO, sick, remote) |
| 1.2 | Confirm reception coverage for main line |
| 1.3 | Confirm backup if Receptionist is out |
| 1.4 | Notify staff of any coverage gaps |

### 2. Front Desk Readiness

| Step | Action |
|------|--------|
| 2.1 | Unlock main entrance; disarm alarm if applicable |
| 2.2 | Ensure visitor log and sign-in materials are ready |
| 2.3 | Check waiting area: clean, stocked (water, tissues) |
| 2.4 | Verify phone system is operational |
| 2.5 | Confirm main voicemail is cleared or reviewed |

### 3. Mail and Deliveries

| Step | Action |
|------|--------|
| 3.1 | Collect incoming mail from mailbox/lockbox |
| 3.2 | Sort and distribute to appropriate staff |
| 3.3 | Log any overnight deliveries |
| 3.4 | Prepare outgoing mail for pickup |

### 4. Supply Check

| Step | Action |
|------|--------|
| 4.1 | Quick check: paper, toner, staples, envelopes |
| 4.2 | Restock front desk supplies (business cards, intake forms) |
| 4.3 | Reorder if below threshold (via Office Manager process) |

### 5. Facilities

| Step | Action |
|------|--------|
| 5.1 | Walk common areas: clean, lights on, HVAC comfortable |
| 5.2 | Check conference rooms: clean, projector/screen ready |
| 5.3 | Report any maintenance issues to facilities or COO |

### 6. Staff Coordination (as needed)

| Step | Action |
|------|--------|
| 6.1 | Share daily announcements (visitors, meetings, events) |
| 6.2 | Confirm any special arrangements (catering, client visits) |
| 6.3 | Address any immediate staff requests |

---

## Escalation

| Issue | Escalate To |
|-------|-------------|
| No reception coverage | COO, or assign temporary coverage |
| Facility emergency | COO, facilities vendor |
| Security concern | COO, IT/Cybersecurity Lead |

---

## Tools

- Calendar (Google, Outlook) for coverage and PTO
- Phone system admin for voicemail
- Supply inventory or reorder system

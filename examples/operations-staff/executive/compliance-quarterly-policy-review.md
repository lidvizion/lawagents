# Compliance / Risk Manager: Quarterly Policy Review

**Role:** [Compliance / Risk Manager](../../../roles/compliance-risk-manager/README.md)  
**Frequency:** Quarterly  
**Approval:** COO or Managing Partner approves policy changes

---

## Scenario

Review and update firm policies (handbook, confidentiality, security, ethics) and ensure staff acknowledgment.

---

## Step-by-Step

### 1. Policy Inventory

| Step | Action |
|------|--------|
| 1.1 | List all active policies (handbook, confidentiality, remote work, PTO, etc.) |
| 1.2 | Note last review date for each |
| 1.3 | Identify policies due for refresh (regulatory changes, firm changes) |

### 2. Regulatory and Best Practice Check

| Step | Action |
|------|--------|
| 2.1 | Review bar rules, HIPAA, GDPR (if applicable) for updates |
| 2.2 | Check employment law changes (state, federal) |
| 2.3 | Review cybersecurity guidance (ABA, state bar) |
| 2.4 | Document any required policy updates |

### 3. Draft Updates

| Step | Action |
|------|--------|
| 3.1 | Draft policy revisions |
| 3.2 | Route to COO or legal counsel for review |
| 3.3 | Incorporate feedback |
| 3.4 | Obtain COO/Managing Partner approval |

### 4. Rollout and Acknowledgment

| Step | Action |
|------|--------|
| 4.1 | Publish updated policies (intranet, handbook) |
| 4.2 | Notify staff of changes and effective date |
| 4.3 | Require acknowledgment (signature or e-sign) |
| 4.4 | Track acknowledgments; follow up with non-responders |
| 4.5 | Retain acknowledgment records |

### 5. Training (if significant changes)

| Step | Action |
|------|--------|
| 5.1 | Schedule training session or webinar |
| 5.2 | Record attendance |
| 5.3 | Document training completion |

### 6. Documentation

| Step | Action |
|------|--------|
| 6.1 | Log review date and changes in compliance tracker |
| 6.2 | Report to COO: policies reviewed, updates made, acknowledgments |
| 6.3 | Set next quarter's review focus |

---

## Policy Categories to Review

| Category | Examples |
|----------|----------|
| Employment | Handbook, PTO, remote work, performance |
| Confidentiality | Client data, matter confidentiality |
| Security | Passwords, MFA, device use, clean desk |
| Ethics | Conflicts, solicitation, advertising |
| Data privacy | HIPAA, GDPR, client data handling |

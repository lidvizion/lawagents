# Billing Manager: Monthly Invoice Run

**Role:** [Billing / Finance Manager](../../../roles/billing-finance-manager/README.md)  
**Frequency:** Monthly (e.g., 1st–5th business day for prior month)  
**Approval:** Partner approves invoices before send; COO for write-offs

---

## Scenario

Generate and deliver client invoices for the prior month. Ensure accuracy, attorney review, and timely delivery.

---

## Prerequisites

- All time entries for prior month submitted and approved
- Matter rates and billing guidelines current in Clio (or firm system)
- Engagement terms documented (hourly, contingent, hybrid)

---

## Step-by-Step

### 1. Time Entry Audit (Day 1)

| Step | Action | Tool |
|------|--------|------|
| 1.1 | Run missing time report | Clio, or firm PMS |
| 1.2 | Follow up with attorneys who have not submitted | Email, Slack |
| 1.3 | Review narrative quality; request revisions if needed | Clio |
| 1.4 | Confirm all entries approved by matter responsible attorney | Clio |

### 2. Invoice Generation (Day 2)

| Step | Action | Output |
|------|--------|--------|
| 2.1 | Generate draft invoices for prior month | Draft invoices |
| 2.2 | Apply client billing guidelines (e.g., task codes, rate caps) | Per engagement |
| 2.3 | Flag any write-downs or non-standard entries | For partner review |
| 2.4 | Export or route drafts to responsible attorneys | Clio, email |

### 3. Attorney Review (Day 2–3)

| Step | Action |
|------|--------|
| 3.1 | Send draft invoices to responsible attorneys |
| 3.2 | Allow 24–48 hours for review |
| 3.3 | Incorporate revisions (narrative changes, write-offs) |
| 3.4 | Obtain approval (verbal, email, or system sign-off) |

### 4. Finalize and Deliver (Day 4)

| Step | Action |
|------|--------|
| 4.1 | Finalize approved invoices |
| 4.2 | Send per engagement terms (email, client portal, mail) |
| 4.3 | Log delivery and track acknowledgment where possible |
| 4.4 | Update AR and aging |

### 5. Post-Run Tasks (Day 5)

| Step | Action |
|------|--------|
| 5.1 | Produce aging report; distribute to partners |
| 5.2 | Flag accounts 60+ days for collection follow-up |
| 5.3 | Document any write-offs for partner approval |
| 5.4 | Notify COO that billing close is complete |

---

## Handoffs

| From | To | Item |
|------|-----|------|
| Attorneys | Billing Manager | Time entries, invoice approval |
| Billing Manager | Partners | Draft invoices for review |
| Billing Manager | Clients | Final invoices |
| Billing Manager | COO | Billing close complete, aging report |

---

## Agent-Ready Tasks

| Task | Guardrail |
|------|-----------|
| Invoice generation | Partner approves before send |
| Aging reports | Partner/COO use for collection decisions |
| Write-off analysis | Partner approval required |

---

## Tools

- **Clio** — Time entries, invoices, matter rates, client billing guidelines
- **QuickBooks / Xero** — AR posting, reconciliation (if integrated)

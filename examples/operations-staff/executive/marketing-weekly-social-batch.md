# Marketing Specialist: Weekly Social Content Batch

**Role:** [Marketing Specialist](../../../roles/marketing-specialist/README.md)  
**Frequency:** Weekly (e.g., Friday for following week)  
**Approval:** Low—routine; ensure ethics compliance (no testimonials, solicitation)

---

## Scenario

Plan, create, and schedule social media content across multiple channels for the coming week. Use Planable, Buffer, or Hootsuite for multi-channel posting.

---

## Channels (Typical for Law Firms)

| Channel | Use When |
|---------|----------|
| LinkedIn | Thought leadership, firm news, hiring |
| Facebook | Community, local events, practice area tips |
| X (Twitter) | Quick updates, legal news, engagement |
| Instagram | Culture, office, events (visual) |
| Google Business Profile | Local SEO, reviews, hours |

---

## Step-by-Step

### 1. Content Planning (Day 1)

| Step | Action |
|------|--------|
| 1.1 | Review content calendar for themes (e.g., estate planning month) |
| 1.2 | Identify 5–7 post ideas (blog links, tips, firm news, events) |
| 1.3 | Check for bar rule compliance (no testimonials, no solicitation) |
| 1.4 | Draft copy for each post; adapt per channel (LinkedIn longer, X shorter) |

### 2. Asset Preparation

| Step | Action |
|------|--------|
| 2.1 | Gather images, graphics, or video clips |
| 2.2 | Ensure branding consistency (logo, colors) |
| 2.3 | Optimize for each platform (sizes, aspect ratios) |

### 3. Scheduling in Tool

| Step | Action | Tool |
|------|--------|------|
| 3.1 | Log into Planable, Buffer, or Hootsuite | Planable, Buffer, Hootsuite |
| 3.2 | Create posts for each channel | Per tool |
| 3.3 | Set publish times (e.g., Tue/Thu 9 AM, Wed 12 PM) | Per tool |
| 3.4 | Use approval workflow if required (e.g., partner sign-off) | Per tool |
| 3.5 | Schedule for the week | Per tool |

### 4. CRM and Attribution (Optional)

| Step | Action |
|------|--------|
| 4.1 | Log campaigns in Clio Grow or CRM |
| 4.2 | Tag leads that mention social as source |
| 4.3 | Track engagement (likes, shares, clicks) for reporting |

### 5. Weekly Review

| Step | Action |
|------|--------|
| 5.1 | Review prior week performance |
| 5.2 | Note top-performing posts |
| 5.3 | Adjust next week's mix based on engagement |

---

## Ethics Compliance Checklist

- [ ] No client testimonials or endorsements
- [ ] No improper solicitation (check state bar rules)
- [ ] Disclaimers where required (e.g., "Not legal advice")
- [ ] No confidential or privileged information

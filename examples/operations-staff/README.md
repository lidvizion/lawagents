## Agent-Ready Examples

Examples that can be automated with approval guardrails:

- **Billing:** Invoice generation (partner approval before send)
- **Intake:** CRM data entry, consultation scheduling
- **Records:** Matter folder creation (attorney confirms matter details)
- **Marketing:** Social post scheduling (MCP via Buffer/Hootsuite)

See [AGENT_INSTRUCTIONS.md](../../AGENT_INSTRUCTIONS.md) for the agent-ready task pattern.

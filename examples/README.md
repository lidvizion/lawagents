# Operations Staff Examples

**Purpose:** Practical, step-by-step workflow examples for non-attorney support staff. Use these as templates, checklists, and training materials for law firm operations teams.

**Scope:** All support staff and non-attorney roles—executive/management, legal support, and intake/admin.

## How to Use

1. **Training:** Share examples with new hires as role-specific onboarding.
2. **Checklists:** Print or bookmark for daily/weekly routines.
3. **Process improvement:** Adapt steps to your firm's tools (Clio, MyCase, etc.).
4. **Agent workflows:** Examples marked *Agent-ready* can be automated with approval guardrails.

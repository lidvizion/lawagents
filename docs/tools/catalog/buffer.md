# Buffer

**Category:** Social media scheduling  
**Use when:** Small law firm needs simple multi-channel posting; want AI agent (Claude, Cursor) to schedule via MCP.

| Field | Value |

---

## When to Use

| Scenario | Use Buffer |
|----------|------------|
| AI agent schedules posts (Claude, Cursor) | ✅ Yes (Zapier MCP) |
| Simple queue; add to queue or send now | ✅ Yes |
| Facebook, X, LinkedIn, Instagram, Pinterest, TikTok, Threads, YouTube | ✅ Yes |
| Pause/unpause channel queues | ✅ Yes |
| Create ideas (MCP) | ✅ Yes |
| Complex approval workflows | ⚠️ Consider Planable, Hootsuite |
| Enterprise; multi-brand | ⚠️ Consider Hootsuite |

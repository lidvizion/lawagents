# Tool Catalog

Individual tool files. Each includes:

- **Overview** — Product, best for, pricing, API/MCP
- **When to use** — Scenario table (✅ / ⚠️ / ❌)
- **Score** — Composite from sync sources; sentiment
- **Sync sources** — URLs for G2, Capterra, vendor, Reddit
- **Key features**
- **Related** — Links to alternatives, practice areas

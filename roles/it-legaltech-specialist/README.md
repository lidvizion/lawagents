# IT Administrator / LegalTech Specialist

**Category:** Executive and Management Staff  
**Approval Level:** Low–Medium—routine operations; major changes need approval

---

## Role Summary

The IT Administrator manages software and hardware support for the firm. They maintain firm software (Clio, email, network), troubleshoot issues, and oversee cloud deployments. They have hands-on knowledge of legal SaaS and support attorneys and staff with technology. Reports to IT/Cybersecurity Lead or COO.

---

## Detailed Responsibilities

### 1. Software and Application Support

- **Practice management:** Clio, MyCase, or similar—user support, configuration
- **Email and collaboration:** Google Workspace, Microsoft 365—accounts, permissions
- **Document management:** NetDocuments, iManage, Clio—access, troubleshooting
- **Billing:** Clio billing, QuickBooks—support Billing Manager
- **Troubleshooting:** Resolve user issues; escalate to vendor when needed

### 2. Hardware and Network

- **Workstations:** PCs, Macs; setup, maintenance, replacement
- **Network:** LAN, Wi-Fi, VPN; connectivity support
- **Printers and peripherals:** Setup, troubleshooting
- **Mobile devices:** MDM if applicable; device support

### 3. Cloud and Infrastructure

- **Cloud deployments:** AWS, Vercel, or similar—if firm hosts custom apps
- **SaaS administration:** Admin consoles for Google, Microsoft, Clio, etc.
- **Backups:** Verify backup jobs; support recovery
- **Monitoring:** Infrastructure monitoring; alert on issues

### 4. Support and Ticketing

- **Support ticketing:** Track requests; resolve or escalate
- **Documentation:** Maintain runbooks, FAQs for common issues
- **Onboarding:** Set up new user accounts, email, system access
- **Offboarding:** Revoke access; retrieve devices
